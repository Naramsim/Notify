Notify

Simple API for display beautiful toasts

View project on GitHub

What is this?

Notify is a simple and lightweight library that helps you to display beautiful toasts. It is inspired by the new concept that Android is pushing on Marshmallow OS, which can be see here: Material Android

Can I see a demo?

Of course you can. Try with the buttons below or press "n" on the keyboard

How it works?

Link the script and the css.

	
<script type="text/javascript" src="notify.js"></script>
<script type="text/javascript" src="test_functions.js"></script>

Initialize Notify creating a Toast object.

	
<script type="text/javascript">
	var myToast = new Toast();
	myToast.stayOnFor(2100);
</script>

Then attach the event to something, like pressing buttons, waiting for some download...

	
myToast.start("Hi, page just loaded", "Browse now");
myToast.start("HINT", "Description", "https://url-of-img.jpg", callback, [parameter, other-parameter, ...] );
